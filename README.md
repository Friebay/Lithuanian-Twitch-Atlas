# Lithuanian-Twitch-Atlas
Variuos Lithuanian Twitch communities statistics

This project is inspired by the [Twitch Atlas](https://github.com/KiranGershenfeld/VisualizingTwitchCommunities).

1. Run streamers\get_twitchers.py to get all streamers from twitchers.lt

2. Run streamers\get_twitch_streamers.py to get all IDs of the Twitch streamers

3. Run streamers\get_ivr_fi_log.py to get data about the streamers

4. Run streamers\get_active_streamers.py to get the streamers that stream in the last 14 days and save config.json for rustlog
